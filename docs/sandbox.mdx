# Sandbox lifecycle

Sandboxes stay running until their timeout expires. In the Go SDK, both `SandboxOpts.TimeoutMs` and `sandbox.SetTimeout(...)` expect milliseconds.

For longer-lived workflows, see [pause and resume](/docs/sandbox/persistence) and [auto resume](/docs/sandbox/auto-resume).

## Create a sandbox with a timeout

Use `e2b.Create(ctx, template, opts)` to start a sandbox. Pass `""` for the template to use the default `base` runtime.

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	timeoutMs := 60_000

	sandbox, err := e2b.Create(ctx, "", &e2b.SandboxOpts{
		TimeoutMs: &timeoutMs,
	})
	if err != nil {
		log.Fatal(err)
	}
	defer sandbox.Kill(context.Background(), nil)

	log.Println(sandbox.SandboxID)
}
```

## Change the timeout during runtime

Use `sandbox.SetTimeout(ctx, timeoutMs, nil)` to extend or reduce the remaining runtime from the current moment.

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()

	sandbox, err := e2b.Create(ctx, "", nil)
	if err != nil {
		log.Fatal(err)
	}
	defer sandbox.Kill(context.Background(), nil)

	if err := sandbox.SetTimeout(ctx, 30_000, nil); err != nil {
		log.Fatal(err)
	}
}
```

## Retrieve sandbox information

Use `sandbox.GetInfo(ctx, nil)` to read lifecycle details such as the sandbox ID, template ID, metadata, state, and start or end timestamps.

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()

	sandbox, err := e2b.Create(ctx, "", nil)
	if err != nil {
		log.Fatal(err)
	}
	defer sandbox.Kill(context.Background(), nil)

	info, err := sandbox.GetInfo(ctx, nil)
	if err != nil {
		log.Fatal(err)
	}

	log.Printf("sandbox=%s state=%s ends=%s\n", info.SandboxID, info.State, info.EndAt)
}
```

## Shut down a sandbox

Use `sandbox.Kill(ctx, nil)` to stop the sandbox before its timeout is reached.

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()

	sandbox, err := e2b.Create(ctx, "", nil)
	if err != nil {
		log.Fatal(err)
	}

	if err := sandbox.Kill(ctx, nil); err != nil {
		log.Fatal(err)
	}
}
```
