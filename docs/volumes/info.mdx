# File and Directory Metadata

Use these methods on `*e2b.Volume`:

- `GetInfo(ctx, path, opts)`
- `Exists(ctx, path, opts)`
- `UpdateMetadata(ctx, path, metadata, opts)`

## Get file metadata

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	volume, _ := e2b.CreateVolume(ctx, "my-volume", nil)

	_, _ = volume.WriteFile(ctx, "/test_file.txt", "Hello, world!", nil)

	info, err := volume.GetInfo(ctx, "/test_file.txt", nil)
	if err != nil {
		log.Fatal(err)
	}

	log.Println(
		info.Name,
		info.Type,
		info.Path,
		info.Size,
		info.Mode,
		info.UID,
		info.GID,
		info.Atime,
		info.Mtime,
		info.Ctime,
	)
}
```

## Get directory metadata

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	volume, _ := e2b.CreateVolume(ctx, "my-volume", nil)

	_, _ = volume.MakeDir(ctx, "/test_dir", nil)

	info, err := volume.GetInfo(ctx, "/test_dir", nil)
	if err != nil {
		log.Fatal(err)
	}

	log.Println(info.Name, info.Type, info.Path)
}
```

## Check whether a path exists

`Exists(...)` returns `false` on not-found instead of raising an error.

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	volume, _ := e2b.CreateVolume(ctx, "my-volume", nil)

	exists, err := volume.Exists(ctx, "/test_file.txt", nil)
	if err != nil {
		log.Fatal(err)
	}

	log.Println(exists)
}
```

## Update metadata

Use `VolumeMetadataOptions` to update `UID`, `GID`, and `Mode`.

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	volume, _ := e2b.CreateVolume(ctx, "my-volume", nil)

	_, _ = volume.WriteFile(ctx, "/test_file.txt", "Hello, world!", nil)

	uid := 1000
	gid := 1000
	mode := 0o600

	updated, err := volume.UpdateMetadata(ctx, "/test_file.txt", &e2b.VolumeMetadataOptions{
		UID:  &uid,
		GID:  &gid,
		Mode: &mode,
	}, nil)
	if err != nil {
		log.Fatal(err)
	}

	log.Println(updated.Name, updated.Mode, updated.UID, updated.GID)
}
```
