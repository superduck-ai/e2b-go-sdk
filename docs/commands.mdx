# Running Commands in Sandbox

In the Go SDK, use `sandbox.Commands.Run(ctx, cmd, opts)` to execute terminal commands inside a sandbox.

## Go-specific behavior

- `Run(...)` returns a runtime-polymorphic value
- when `opts == nil` or `opts.Background == false`, it returns `*e2b.CommandResult`
- when `opts.Background == true`, it returns `*e2b.CommandHandle`
- callers usually type-assert the returned value after the call

## Run a command

```go
package main

import (
	"context"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	sandbox, _ := e2b.Create(ctx, "", nil)

	execution, _ := sandbox.Commands.Run(ctx, "ls -l", nil)
	result := execution.(*e2b.CommandResult)

	_ = result.ExitCode
	_ = result.Stdout
	_ = result.Stderr
}
```

`CommandResult` includes:

- `ExitCode`
- `Stdout`
- `Stderr`
- `Error`

For background commands, see [Run Commands in Background](/docs/commands/background). For streaming stdout and stderr, see [Streaming Command Output](/docs/commands/streaming).
