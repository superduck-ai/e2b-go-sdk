# Install Custom Packages

There are two main ways to install packages with the Go SDK:

1. build a custom template with packages preinstalled
2. install packages at sandbox runtime with commands

## Create a custom sandbox template

Use this when you know the dependencies ahead of time.

### Template definition

```go
package main

import (
	e2b "github.com/superduck-ai/e2b-go-sdk"
)

var template = e2b.Template(nil).
	FromBaseImage().
	PipInstall([]string{"cowsay"}, nil).
	NpmInstall([]string{"cowsay"}, nil)
```

### Build script

```go
package main

import (
	"context"
	"log"

	"github.com/joho/godotenv"
	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	_ = godotenv.Load()

	_, err := e2b.Build(context.Background(), template, "custom-packages", &e2b.BuildOptions{
		BasicBuildOptions: e2b.BasicBuildOptions{
			CpuCount:    2,
			MemoryMB:    2048,
			OnBuildLogs: e2b.DefaultBuildLogger(),
		},
	})
	if err != nil {
		log.Fatal(err)
	}
}
```

### Use the custom template

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()

	sandbox, err := e2b.Create(ctx, "custom-packages", nil)
	if err != nil {
		log.Fatal(err)
	}
	defer sandbox.Kill(context.Background(), nil)
}
```

## Install packages during sandbox runtime

Use this when you do not know the dependencies beforehand.

These installs affect only the current sandbox instance.

### Install Python packages with pip

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	sandbox, err := e2b.Create(ctx, "", nil)
	if err != nil {
		log.Fatal(err)
	}
	defer sandbox.Kill(context.Background(), nil)

	_, err = sandbox.Commands.Run(ctx, "pip install cowsay", nil)
	if err != nil {
		log.Fatal(err)
	}

	execution, err := sandbox.Commands.Run(ctx, `python3 -c "import cowsay; print(cowsay.get_output_string('cow', 'Hello, world!'))"`, nil)
	if err != nil {
		log.Fatal(err)
	}
	result := execution.(*e2b.CommandResult)

	log.Println(result.Stdout)
}
```

### Install Node.js packages with npm

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	sandbox, err := e2b.Create(ctx, "", nil)
	if err != nil {
		log.Fatal(err)
	}
	defer sandbox.Kill(context.Background(), nil)

	_, err = sandbox.Commands.Run(ctx, "npm install cowsay", nil)
	if err != nil {
		log.Fatal(err)
	}

	execution, err := sandbox.Commands.Run(ctx, `node -e "const cowsay = require('cowsay'); console.log(cowsay.say({ text: 'Hello, world!' }))"`, nil)
	if err != nil {
		log.Fatal(err)
	}
	result := execution.(*e2b.CommandResult)

	log.Println(result.Stdout)
}
```

### Install Debian packages with apt

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	sandbox, err := e2b.Create(ctx, "", nil)
	if err != nil {
		log.Fatal(err)
	}
	defer sandbox.Kill(context.Background(), nil)

	_, err = sandbox.Commands.Run(ctx, "apt-get update && apt-get install -y curl git", &e2b.CommandStartOpts{
		User: "root",
	})
	if err != nil {
		log.Fatal(err)
	}
}
```
