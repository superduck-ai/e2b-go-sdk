# Auto-resume on Request

Auto-resume lets a paused sandbox wake up when activity reaches it.

Configure it through `SandboxOpts.Lifecycle`:

- set `OnTimeout: "pause"`
- set `AutoResume` to `true`

`AutoResume` can only be enabled when the resolved `OnTimeout` is `"pause"`.

## Configure auto-resume

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	timeoutMs := 10 * 60 * 1000
	autoResume := true

	sandbox, err := e2b.Create(ctx, "", &e2b.SandboxOpts{
		TimeoutMs: &timeoutMs,
		Lifecycle: &e2b.SandboxLifecycle{
			OnTimeout:  "pause",
			AutoResume: &autoResume,
		},
	})
	if err != nil {
		log.Fatal(err)
	}
	defer sandbox.Kill(context.Background(), nil)
}
```

## SDK activity can wake a paused sandbox

With auto-resume enabled, a later SDK call can resume the sandbox without an explicit `Connect(...)` first.

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	timeoutMs := 10 * 60 * 1000
	autoResume := true

	sandbox, err := e2b.Create(ctx, "", &e2b.SandboxOpts{
		TimeoutMs: &timeoutMs,
		Lifecycle: &e2b.SandboxLifecycle{
			OnTimeout:  "pause",
			AutoResume: &autoResume,
		},
	})
	if err != nil {
		log.Fatal(err)
	}
	defer sandbox.Kill(context.Background(), nil)

	_, err = sandbox.Files.Write(ctx, "/home/user/hello.txt", "hello from go", nil)
	if err != nil {
		log.Fatal(err)
	}
	if _, err := sandbox.Pause(ctx, nil); err != nil {
		log.Fatal(err)
	}

	value, err := sandbox.Files.Read(ctx, "/home/user/hello.txt", nil)
	if err != nil {
		log.Fatal(err)
	}
	info, err := sandbox.GetInfo(ctx, nil)
	if err != nil {
		log.Fatal(err)
	}

	content := value.(string)
	log.Println(content, info.State)
}
```

## Incoming traffic can wake a paused sandbox

This is useful for preview URLs and web services.

```go
package main

import (
	"context"
	"log"
	"net/http"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	timeoutMs := 5 * 60 * 1000
	autoResume := true

	sandbox, err := e2b.Create(ctx, "", &e2b.SandboxOpts{
		TimeoutMs: &timeoutMs,
		Lifecycle: &e2b.SandboxLifecycle{
			OnTimeout:  "pause",
			AutoResume: &autoResume,
		},
	})
	if err != nil {
		log.Fatal(err)
	}
	defer sandbox.Kill(context.Background(), nil)

	execution, err := sandbox.Commands.Run(ctx, "python3 -m http.server 3000", &e2b.CommandStartOpts{
		Background: true,
	})
	if err != nil {
		log.Fatal(err)
	}

	handle := execution.(*e2b.CommandHandle)
	defer handle.Kill()

	url := "https://" + sandbox.GetHost(3000)
	req, err := http.NewRequestWithContext(ctx, http.MethodGet, url, nil)
	if err != nil {
		log.Fatal(err)
	}
	if sandbox.TrafficAccessToken != "" {
		req.Header.Set("e2b-traffic-access-token", sandbox.TrafficAccessToken)
	}

	log.Println(req.URL.String())
}
```
