# Sandbox Persistence

Pausing a sandbox preserves its filesystem and memory state so you can resume it later.

The main lifecycle calls are:

- `sandbox.Pause(ctx, nil)`
- `sandbox.Connect(ctx, nil)` or `e2b.Connect(ctx, sandboxID, nil)`
- `sandbox.Kill(ctx, nil)` or `e2b.Kill(ctx, sandboxID, nil)`

## Pause, resume, and kill a sandbox

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	sandbox, err := e2b.Create(ctx, "", nil)
	if err != nil {
		log.Fatal(err)
	}

	paused, err := sandbox.Pause(ctx, nil)
	if err != nil {
		log.Fatal(err)
	}
	log.Println("paused:", paused)

	sameSandbox, err := sandbox.Connect(ctx, nil)
	if err != nil {
		log.Fatal(err)
	}
	log.Println("resumed:", sameSandbox.SandboxID)

	if err := sandbox.Kill(ctx, nil); err != nil {
		log.Fatal(err)
	}
}
```

`Pause(...)` returns `false` when the sandbox was already paused.

## List paused sandboxes

Use the `State` filter with `e2b.SandboxState("paused")`.

```go
package main

import (
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	paginator := e2b.List(&e2b.SandboxListOpts{
		Query: &struct {
			Metadata map[string]string
			State    []e2b.SandboxState
		}{
			State: []e2b.SandboxState{e2b.SandboxState("paused")},
		},
	})

	sandboxes, err := paginator.NextItems()
	if err != nil {
		log.Fatal(err)
	}

	log.Println(len(sandboxes))
}
```

## Remove a paused sandbox

You can terminate a paused sandbox through either the instance method or the package-level helper.

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	sandbox, err := e2b.Connect(ctx, "sbx_123", nil)
	if err != nil {
		log.Fatal(err)
	}

	if err := sandbox.Kill(ctx, nil); err != nil {
		log.Fatal(err)
	}

	deleted, err := e2b.Kill(ctx, "sbx_123", nil)
	if err != nil {
		log.Fatal(err)
	}

	log.Println(deleted)
}
```

## Set the timeout when reconnecting

`SandboxConnectOpts.TimeoutMs` uses milliseconds.

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	timeoutMs := 60_000

	sandbox, err := e2b.Connect(ctx, "sbx_123", &e2b.SandboxConnectOpts{
		TimeoutMs: &timeoutMs,
	})
	if err != nil {
		log.Fatal(err)
	}

	log.Println(sandbox.SandboxID)
}
```

## Auto-pause on timeout

Configure auto-pause through `SandboxOpts.Lifecycle`.

```go
package main

import (
	"context"
	"log"

	e2b "github.com/superduck-ai/e2b-go-sdk"
)

func main() {
	ctx := context.Background()
	timeoutMs := 10 * 60 * 1000
	autoResume := false

	sandbox, err := e2b.Create(ctx, "", &e2b.SandboxOpts{
		TimeoutMs: &timeoutMs,
		Lifecycle: &e2b.SandboxLifecycle{
			OnTimeout:  "pause",
			AutoResume: &autoResume,
		},
	})
	if err != nil {
		log.Fatal(err)
	}
	defer sandbox.Kill(context.Background(), nil)
}
```

If a sandbox is killed, it is permanently deleted and cannot be resumed.
