# Create charts & visualizations

The current Go SDK does not automatically turn Matplotlib output into typed chart results.

Instead, generate either:

- static image files such as PNG or SVG
- structured JSON files describing the chart data

Then read those artifacts back through `sandbox.Files`.

See:

- [Static charts](/docs/code-interpreting/create-charts-visualizations/static-charts)
- [Interactive charts](/docs/code-interpreting/create-charts-visualizations/interactive-charts)
